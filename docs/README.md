# Dokumentation (agent-control-surface)

Diese Sammlung bündelt die Projekt-Dokumentation an einem Ort.

## Inhalte

- [Blaupause / Zielbild](blaupause.md)
  - Architektur- und Prinzipien-Dokument („Orchestrator + Gedächtnisanker“).
  - Dient als Leitlinie für Roadmap und Priorisierung.
- [RUNBOOK (Betrieb)](../RUNBOOK.md)
  - Installation, Start, SSH-Tunnel, systemd, Troubleshooting.

## Konvention

- Operative Anleitungen gehören ins RUNBOOK.
- Architektur-/Zielbild-Themen gehören in die Blaupause.
